# OpenAlbion

Welcome to the official documentation for OpenAlbion, maintained by the Albion College EECS Club.

## What is OpenAlbion?

OpenAlbion is an open data portal designed to provide transparent access to Albion College's institutional research, faculty information, and educational metrics. Our mission is to empower the Albion College community and researchers with reliable, up-to-date information about the institution.

## Quick Start

To begin using OpenAlbion's API:

<Steps>
### Log in
Log into your [dashboard](https://openalbion.org/dashboard) 

### API key
Register for an API key in your profile settings

### Configure your request
Check out your endpoint's documentation and configure your request to that endpoint
</Steps>

## FAQ

<details>
  <summary>**What data does OpenAlbion provide?**</summary>
    OpenAlbion provides access to a variety of institutional data including faculty information, course metrics, research publications, and enrollment statistics.
</details>

<details>
  <summary>**How often is the data updated?**</summary>
      Most datasets are updated on a semester basis, with some high-frequency metrics updated weekly or monthly. Each dataset includes a "last updated" timestamp so you can verify the recency of the information.
</details>

<details>
  <summary>**Is there a rate limit for API requests?**</summary>
   There is no API rate limit at this time.
</details>

<details>
  <summary>**How do I report an issue with the data?**</summary>
    If you believe you've found an error in our data or have concerns about data quality, please email us at contact@openalbion.org with details about the problem.
</details>
